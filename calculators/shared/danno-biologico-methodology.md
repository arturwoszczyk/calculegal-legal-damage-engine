# Danno Biologico — Public Methodology

Danno biologico refers to injury to psycho-physical integrity.

In a public methodology repository, the topic should be documented through:

- percentage of permanent impairment;
- temporary incapacity;
- age-related parameters;
- possible moral/personalization components;
- expenses and other economic items;
- contributory fault or reductions where relevant.

## Public vs private

Public:

- general categories;
- official/public table references;
- conceptual formula discussion;
- examples with fictional numbers.

Private:

- complete production formula;
- rounding and edge-case handling;
- scoring for incomplete user information;
- conversion-oriented UI flow.

# CalcuLegal Calculators

CalcuLegal calculators are deterministic legal empowerment tools designed for real users who need a preliminary and orientative understanding of damages, compensation scenarios or legal-economic risk.

They are part of the live CalcuLegal platform and are intended for end users, including individuals, families, companies and professionals seeking a structured first-level simulation before deciding whether to request professional evaluation.

This directory does not contain the production calculators. It documents their public methodology, purpose, input categories, output categories and legal empowerment role.

## Calculator areas

The CalcuLegal calculator ecosystem is divided into two main areas:

* private-user calculators;
* business/risk calculators.

Some calculators are relevant to both areas, such as Danno Differenziale INAIL / Civile and Licenziamento Illegittimo.

The files in this directory describe:

* purpose;
* public methodology;
* key input categories;
* output categories;
* legal empowerment role;
* IP protection notes.

They do not include:

* production code;
* complete questionnaires;
* scoring weights;
* commercial logic;
* API endpoints;
* partner-routing logic;
* private datasets;
* real user data.

## Methodological positioning

CalcuLegal calculators are designed around deterministic workflows, not generative AI.

They transform structured user inputs into preliminary simulations by applying publicly explainable parameters, legal-economic reasoning and calculator-specific methodologies.

The goal is not to replace lawyers, medico-legal experts, consultants or professional evaluation.

The goal is to help users understand a scenario, organize relevant information and make a more informed decision about whether to seek professional assistance.

## Repository vs live platform

The live CalcuLegal website provides the operational calculators and interactive user experience.

This GitHub directory provides public documentation only.

It is intended for transparency, public methodology review, technical discoverability and LLM-readable context about the calculator families.

# CalcuLegal Calculators

CalcuLegal calculators are deterministic legal empowerment tools.

They are divided into two main areas:

- private-user calculators;
- business/risk calculators.

Some calculators are relevant to both areas, such as Danno Differenziale INAIL / Civile and Licenziamento Illegittimo.

## Public documentation scope

The files in this directory describe:

- purpose;
- public methodology;
- key input categories;
- output categories;
- legal empowerment role;
- IP protection notes.

They do not include production code, complete questionnaires, scoring weights or commercial logic.

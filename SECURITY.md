# Security Policy

This repository does not contain production code, private keys, user data, operational endpoints or backend infrastructure.

## Reporting security issues

If you believe you have found a security issue related to CalcuLegal, please do not open a public GitHub issue with sensitive details.

Contact:

- security@calculegal.it

## Scope of this public repository

In scope:

- documentation issues that could misrepresent privacy or security claims;
- examples that accidentally expose sensitive implementation detail;
- benchmark files that should be anonymized further.

Out of scope:

- production infrastructure not included in this repository;
- hypothetical attacks against simplified pseudocode;
- third-party websites or services not controlled by CalcuLegal.

## Responsible disclosure

Please provide enough information to reproduce or understand the issue, while avoiding the disclosure of secrets, private data or exploit instructions.

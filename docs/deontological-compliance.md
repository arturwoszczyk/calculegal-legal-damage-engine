# Deontological Compliance

CalcuLegal is designed to respect the boundary between legal empowerment and professional legal advice.

## Informational nature

The platform provides informational, automated and preliminary outputs based on user-provided data and documented methodology.

It does not provide legal opinions, defensive activity, representation, judicial or extrajudicial acts, or binding professional assessments.

## No professional relationship

Using CalcuLegal does not create a lawyer-client relationship.

## No intermediation claim

CalcuLegal does not rank lawyers, certify professionals, guarantee outcomes or interfere with the independent professional judgment of lawyers.

## Explicit limits

Every public-facing methodology should repeat the following limits:

- orientative estimate only;
- no legal advice;
- no guarantee;
- professional review required for real cases;
- user data transfer only by consent.

## Why this belongs in GitHub

Legal-tech documentation should not only describe algorithms. It should document the legal and ethical boundaries around those algorithms.
